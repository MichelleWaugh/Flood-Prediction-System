# Flood-Prediction-System
# Flood Probability Prediction using Deep Learning

## Overview

This project uses a Deep Neural Network (DNN) built with TensorFlow and Keras to predict flood probability based on environmental, infrastructure, and socio-economic factors.

The workflow includes:

* Data loading and preprocessing
* Exploratory Data Analysis (EDA)
* Missing value analysis
* Outlier detection
* Feature scaling
* Neural Network model training
* Model evaluation
* Prediction generation for test data
* Submission file creation

The model is trained using the provided training dataset and generates flood probability predictions for unseen test data.

# Machine Learning Workflow

## 1. Data Loading

Datasets are loaded using Pandas:

```python
train_df = pd.read_csv("train.csv")
test_df = pd.read_csv("test.csv")
```

---

## 2. Data Cleaning

Operations performed:

* Remove completely empty columns
* Check missing values
* Generate missing-value statistics

---

## 3. Exploratory Data Analysis (EDA)

Visualizations include:

### Histogram Analysis

Used to understand:

* Distribution of features
* Skewness
* Frequency patterns

### Boxplots

Used to detect:

* Outliers
* Feature spread

### Correlation Heatmap

Used to identify:

* Relationships between features
* Feature relevance to FloodProbability

---

## 4. Outlier Detection

Uses the IQR (Interquartile Range) method.

Formula:

```text
Lower Bound = Q1 - 1.5 × IQR

Upper Bound = Q3 + 1.5 × IQR
```

Outliers outside these limits are identified and removed.

---

## 5. Data Scaling

StandardScaler is used.

```python
StandardScaler()
```

Benefits:

* Faster convergence
* Better neural network performance
* Uniform feature scaling

---

## 6. Train-Test Split

Dataset is divided into:

### Training Set

Used for learning patterns.

### Validation Set

Used for tuning.

### Testing Set

Used for final evaluation.

Split:

```python
70% Train
20% Validation
30% Test
```

---

## 7. Deep Neural Network Architecture

Model Architecture:

```text
Input Layer
      ↓
Dense (175 neurons, ReLU)
      ↓
Dense (75 neurons, ReLU)
      ↓
Dense (25 neurons, ReLU)
      ↓
Dense (1 neuron, Sigmoid)
```

---

## Activation Functions

### ReLU

Used in hidden layers.

```python
activation="relu"
```

### Sigmoid

Used in output layer.

```python
activation="sigmoid"
```

Produces output between:

```text
0 and 1
```

representing flood probability.

## How to Run the Project in Jupyter Notebook
Step 1: Install Python

Download Python:

Python Official Website

Recommended:

Python 3.10 or Python 3.11

Step 2: Create Virtual Environment

Open Command Prompt:

mkdir FloodPrediction
cd FloodPrediction

python -m venv tfenv

Activate:

Windows
tfenv\Scripts\activate
Mac/Linux
source tfenv/bin/activate

Step 3: Install Required Libraries
pip install tensorflow keras pandas numpy matplotlib seaborn scikit-learn pillow jupyter

Verify:

pip list
Step 4: Launch Jupyter Notebook
jupyter notebook

Browser opens automatically.

Step 5: Open Project Folder

Ensure the following files are present:

train.csv
test.csv
flood_prediction.ipynb

Step 6: Run Notebook

Execute cells sequentially:

Cell → Run All or

Shift + Enter for each cell.

Step 7: Generate Predictions

After execution:

sample_submission.csv will be generated in the project folder.
